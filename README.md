# 休憩時間管理タイマー
このタイマーには、作業時間を計測するストップウォッチと、休憩時間を計測するタイマーの2つの機能があります。
作業をした分だけ休憩時間が増える仕組みになっており、適切な休憩を取るのに役立ちます。（厳密には、リセットボタンを押すことで作業時間が休憩時間に加算されます。）
